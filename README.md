# GenAI Data Validation Assistant

This project demonstrates how Generative AI concepts can be applied to data validation workflows.

## Overview
The tool analyzes a dataset and identifies common data quality issues such as missing values and duplicate records. It then provides automated suggestions inspired by AI-based approaches for improving data quality.

## Features
- Detects missing values in datasets
- Identifies duplicate rows
- Generates suggestions for data cleaning and validation
- Simple and extensible Python-based workflow

## Tech Stack
- Python
- pandas
